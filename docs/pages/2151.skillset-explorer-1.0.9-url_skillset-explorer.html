Use & Share Without Limits

The Skill Set is shared as Creative Commons CC0 Universal (Public Domain Dedication).

Works Well With SSI

Each Skill has its own ID (SZI) which can be used to generate a DID and with Verifiable Credentials (VCs).

They are available as decentralised assets on the Cardano blockchain and, along with their DID-Document. via the permissioned API.

Community Driven

An extensible set of skills driven by community governance that work with skillzeb templates i.e. Skills Required / Gained.
