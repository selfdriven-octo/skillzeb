Learner Driven & Growth Focused Learning

Allowing learning-partners to collaborate, without necessarily knowing each other, on supporting learners build their skills.

Measuring By Activity

Measuring growth using a common set of skills.

Recognising that ...

The foundation:

Learning happens constantly and anywhere - so we need a common set of skills to enable sharing between the learner and learning-partners.

To speak a common language you need a common set of nouns, in this case skills.

It has to be universal, enabling global learning.

We need to work with intelligent machines, and understand their capability using the same skills set.

What

The skills set is...

Inclusive

Leverages existing skill sets.
Extensible.

Open

Uses the creative Commons CC0 Public Domain Licence.
So the skills set can be freely used by any learning ecosystem.

Inclusive Accessibility

The skill set can be accessed in many ways.
Download, API, Within the app and On-Chain.

On-Chain

The skill set is decentralised on the Cardano blockchain.
Making it immutable and publically accessible.

Skill Set Curation (Additions, Edits etc)

The skill set will eventually be curated by the skillzeb Community, including a DAO (Decentralised Autonomous Organisation), in the short term, as it is being established, the skillzeb Foundation is curating it based on community requests using this online form or the API
