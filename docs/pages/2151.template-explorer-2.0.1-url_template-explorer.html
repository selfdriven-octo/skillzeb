Community Driven

A shared set of templates driven by community contributions that can be integrated into any app, website, framework, project or alike. e.g. selfdriven learning/project templates.

Use With GenAI

Create templates with GenAI based on Starter templates, or using services like heyOcto.ai generate learner specific templates in real-time.

Works Well With SSI

Each Template has its own ID (SZT) which can be used to generate a DID and with Verifiable Credentials (VCs) to establish trust about how the credential was achieved.

The "Template ID" (a BLAKE2b hash of the template JSON definition) is available as a asset on the Cardano blockchain and, along with its DID-Document, via the permissioned API.
